# teambuildr

[teambuildr](https://teambuildr.vercel.app) is a team randomizer with fun animations! The current version works best on large screens.

![Home](https://github.com/francizesperanza/teambuildr/readme_imgs/teambuilder_home.png?raw=true)
![Home](https://github.com/francizesperanza/teambuildr/readme_imgs/teambuilder_sample.png?raw=true)

Features:
- up to 10 teams
- generated team names
- teams can be copied to clipboard
- unlimited members

Coming Soon:
- team leaders
- responsive design
